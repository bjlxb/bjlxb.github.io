后端某个接口运行时间长，需要单独对接口设置超时时间！
<!--more-->
### 全局设置网络超时：
```
axios.defaults.timeout = 30000;
```
### 某个请求单独设置网络超时：
```
let timeout = parseInt(30000);
this.$ajax.post(url, params, {timeout: timeout})
  .then(res => {
    console.log('response='+response);
  })
  .catch(reason => {
    console.log('reason'+reason);
  })
});
```
