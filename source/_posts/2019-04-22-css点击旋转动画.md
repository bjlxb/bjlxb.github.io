验证码点击刷新按钮时,按钮会出现一个旋转效果,写一个简单的例子!
<!--more-->
#### 自动旋转
```css
<html lang="zh-CN">
<head>
    <meta charset="utf-8">
    <title>css点击旋转动画</title>
    <style>
        //css代码
        .header{
            -webkit-animation:rotateImg 1s linear infinite;
            width: 80px ;
            height: 80px;
            border: 1px solid #ccc;
            vertical-align: middle;
        }

        @keyframes rotateImg {
          0% {transform : rotate(0deg);}
          100% {transform : rotate(360deg);}
        }

        @-webkit-keyframes rotateImg {
            0%{-webkit-transform : rotate(0deg);}
          100%{-webkit-transform : rotate(360deg);}
        }
     </style>
</head>
<body>
	//html代码
	<img class="header" src="me.png">
</body>
</html>
```
#### 点击input旋转
限制: 必须用input元素，只有input元素的:focus才有效
```css
<html lang="zh-CN">
<head>
    <meta charset="utf-8">
    <title>css点击旋转动画</title>
    <style>
        input{
            border: 1px solid #dedede;
            background: #fff;
            padding: 5px;
        }
        input:focus{
            outline: none;
            animation: rotatefresh 1s;
        }
        input:active{
            /* 加上这个才可以连续点击 */
            animation: none;  
            background: '#eee';
        }
        @keyframes rotatefresh {
            from { transform: rotate(0deg) }
            to {
                transform: rotate(360deg);
                transition: all 0.6s;
            }
        }
    </style>
</head>
<body>
    <input type="button" value="点击旋转">
</body>
</html>
```

```css
            animation: none;  
            background: '#eee';
        }
        @keyframes rotatefresh {
            from { transform: rotate(0deg) }
            to {
                transform: rotate(360deg);
                transition: all 0.6s;
            }
        }
    </style>
</head>
<body>
    <input type="button" value="点击旋转">
</body>
</html>
```
#### 补充:
```
未访问时的状态（鼠标点击前显示的状态）
a:link{color:#fff}  
 /*鼠标悬停时的状态*/
  a:hover{color:#fff}  
/*已访问过的状态（鼠标点击后的状态）*/
  a:visited{color:#fff}  
/*鼠标点击时的状态*/
  a:active{color:#fff}  
/* 点击后鼠标移开保持鼠标点击时的状态（只有在<a href="#"></a>时标签中有效）*/
  a:focus{color:#fff} 
```
