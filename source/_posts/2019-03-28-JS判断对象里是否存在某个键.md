后台返回数据,有时不判断是否存在该键直接取值会出现报错情况.
<!--more-->
## in关键字
```javascript
var array = {a:1}
console.log("a" in array)
// true
console.log("b" in array)
// false
```
## hasOwnProperty()方法
```javascript
var array = {a:1}
console.log(o.hasOwnProperty("a");)
// true
console.log(o.hasOwnProperty("b");)
// false
```
