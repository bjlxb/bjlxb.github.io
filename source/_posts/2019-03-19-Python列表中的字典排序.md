工作或面试中经常会遇到一些复杂的排序问题，举一些比较常用的排序例子。
<!--more-->
## 按字典value排序
```python
student={
	'language_score': 90,
	'math_score': 95,
	'enlish_score': 80,
	'chemical_score': 65
}
sort = sorted(student.iteritems(), key=lambda d: d[1],reverse=True)
print(sort)
输出：
[('math_score', 95), ('language_score', 90), ('enlish_score', 80), ('chemical_score', 65)]
```

## list中字典的某个key的value排序
```python
# 按照学生成绩排序，取前三名
from operator import itemgetter

student_list = [
{'name': 'zhangsan', 'score': 60},
{'name': 'lisi', 'score': 90},
{'name': 'wangwu', 'score': 80},
{'name': 'zhaoliu', 'score': 70},
{'name': 'tianqi', 'score': 100}
]
sorted(student_list, key=itemgetter('score'), reverse=True)[:3]
sorted(student_list, key=lambda student: student['score'])[-3:]
sorted(student_list, key=lambda stu: stu["score"], reverse=True)[:3]
```
## list中字典的多个key的value排序
```python
student_info = [
{'name':'homer', 'age':39}, 
{'name':'bart', 'age':10}, 
{'name':'aart', 'age':20}, 
{'name':'aart', 'age':10}
]
sort = sorted(student_info, key=lambda student_info : (student_info['name'], -student_info['age']), reverse = True)

print sort

```
