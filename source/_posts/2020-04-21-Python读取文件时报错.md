#### 读取文件报错
```python
UnicodeDecodeError: 'gbk' codec can't decode byte 0x80 in position 205: illegal multibyte sequence
```
###### 原因：
编码问题
###### 解决方案：
1. 读取时加入编码
```python
FILE_OBJECT= open('order.log','r', encoding='UTF-8')
```
2. 直接读字节
```python
FILE_OBJECT= open('order.log','rb')
```
