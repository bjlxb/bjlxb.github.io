记录一下如何临时使用淘宝源
<!--more-->

暂时使用：
```bash
npm --registry https://registry.npm.taobao.org i express
```
永久使用：
```bash
npm conf set registry https://registry.npm.taobao.org
```
安装检测用cnpm -v就可以检测到安装路径等。

通过cnpm使用：
```bash
npm i -g cnpm --registry=https://registry.npm.taobao.org
```
实例使用（npm一律改为cnpm即可） ：
```bash
cnpm i express
```
