os.path模块详解

2020-01-29
Python
  • Python
本文总阅读量次

工作中经常会使用到os.path模块，记录一下基本用法！

os.path.abspath

返回一个目录的绝对路径

os.path.exists

判断指定文件是否存在

os.path.getsize

返回文件的大小

os.path.isdir

判断是否为目录

os.path.isfile

判断是否为文件

os.path.join

join(a, *p)
将目录名和文件的基名拼接成一个完整的路径

os.path.basename

返回一个目录

os.path.dirname

返回一个目录名

os.path.getatime

返回指定文件最后一次的访问时间

os.path.getctime

返回指定文件最后一次的改变时间

os.path.isabs

返回是否为绝对路径

os.path.islink

返回指定参数是否为软链接

os.path.ismount

判断指定参数是否为挂载点

os.path.realpath

返回指定文件的标准路径，而非软链接所在的路径

os.path.samefile

判断两个路径是否指向同一个文件

os.path.split

分割目录名，返回由其目录名和基名给成的元组

os.path.splitext

分割文件名，返回由文件名和扩展名组成的元组
